#AI Notes App

## Features
- Create, read, update, and delete notes (CRUD)
- AI-powered summary for long notes
- Auto-tagging and categorization using AI
- Search notes by keyword or AI-generated tags
- Cloud sync and backup (MongoDB)
- Responsive and mobile-friendly design

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **AI Integration:** OpenAI API (for summarization, tagging)
- **Other Tools:** Axios, dotenv, Cors


---

### 6️⃣ Usage
- Write your notes in the text editor.
- Click "Summarize" to let AI generate a short summary.
- Use "Auto Tag" to categorize notes automatically.
- Search notes using keywords or tags.
